# ビジョンとは
```
私たちは、非現実的で不可能と思われることに挑戦し、現実的で可能なことだと証明したい。
自分が正しいと思ったことをやるべきだ。これだと思ったら、まずやってみる。
（P&G CEO リチャード・デュプリー）
```
ビジョンとは、カラビナで働く人みんなが持つ共通のイメージであり、
**コアバリュー、パーパス、ミッション**で成り立っている
